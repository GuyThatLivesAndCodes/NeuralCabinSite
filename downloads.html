Prefer to build it yourself?

NeuralCabin is GPL-3.0 — the source is on GitHub and the engine has zero math dependencies. Build the desktop app or use neuralcabin_engine as a normal Rust crate.

You will need rustup and Node.js. Tauri builds installers locally.

View source on GitHub ↗ 
# Prerequisites: Rust (rustup.rs) and Node.js
git clone https://github.com/GuyThatLivesAndCodes/NeuralCabin
cd NeuralCabin

npm install
npm --prefix frontend install

# Development mode — opens Tauri window w/ Vite hot-reload
npm run dev

# Production build — installer in src-tauri/target/release/bundle/
npm run tauri -- build
